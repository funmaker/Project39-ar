# Project39-ar
Project39 - AR Character Experience Simulator The Game

## Requirements

- Windows SDK
- LLVM
- CMake
- Python 3
- Ninja
- OpenCV
- SteamVR

Setup OpenCV env vars: https://github.com/twistedfall/opencv-rust#environment-variables

Work in progress, might be incomplete...
